# fortunate

**We, the Fortunate** is an online art project that seeks to bring people together through sharing personal accounts of important experiences in one's life. Whether it is a person that has impacted you, a memory you hold dear, or a lesson you will always remember, this project is meant to provide a space for those stories. If you would like to read or leave a response, you can visit

[wethefortunate.com](https://wethefortunate.com)

Built with React and Supabase.
